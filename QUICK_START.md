## 🍎 iPhone (Need Mac to Build)

**On a Mac computer, run:**

```bash
cd /path/to/grade-goal
npm install
npm run build
npx cap open ios
```

Then in Xcode: Connect iPhone → Press Play ▶️

**No Mac?** Use Codemagic.io (free cloud build)
